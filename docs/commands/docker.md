# 📄 MkDocsコマンド集

## 📝 基本操作

### 新しいプロジェクトファイルの作成
* 新規プロジェクトを作成します。
```
mkdocs new [dirname]
```


### 起動
* ライブリロード対応のドキュメントサーバーを起動します。
```
mkdocs serve
```

### ビルド
* ドキュメントサイトをビルドします。
```
mkdocs build
```

### ヘルプ
* ヘルプメッセージを表示して終了します。
```
mkdocs -h
```

## 🌐 Git関連

### デプロイ
* GitHub Pagesへデプロイ（公開）します。
```
mkdocs gh-deploy
```
