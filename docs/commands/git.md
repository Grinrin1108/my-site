# 🛠️ Git 便利コマンド集

gitのコマンド集です

## 📝 基本操作（日常的に使う）

### 変更をステージングしてコミットする
```Bash
git add .
git commit -m "ここにメッセージを入力"
```

### リモートリポジトリに反映する
```Bash
git push origin main
```

### ログを一行で綺麗に表示する
グラフ形式で、どのブランチがどう分岐しているか視覚的に確認します。
```Bash
git log --oneline --graph --all
```

## 🌿 ブランチ・同期
### ブランチを作成して切り替える
```Bash
git checkout -b feature/new-task
```

### リモートの最新状態を取り込む（プル）
```Bash
git pull origin main
```

## ⚠️ 取り消し・やりなおし
* =="注意：破壊的な操作"==

以下のコマンドは書き換えた内容が消える可能性があるため、実行前に現在の状態を確認してください。

### 直前のコミットメッセージを修正する
``` Bash
git commit --amend -m "修正後のメッセージ"
```

### ファイルの変更をすべて捨てて、最新のコミット状態に戻す
「書き換えたけど、やっぱり最初からやり直したい！」という時に。
```Bash
git restore .
```

### コミットを1つだけ完全に取り消す（戻せないタイプ）
```Bash
git reset --hard HEAD^
```

## 💡 競プロ・自動化で便利なTips

### 特定のファイルだけ無視する（.gitignore）
コンパイル後の .exe ファイルや a.out をGitの管理から外す設定例です。
```Plaintext
*.exe
*.out
.DS_Store
```

### コミットせずに一時的に変更を退避する
別の作業を急ぎでやりたい時に、今の変更を「隠して」おけます。
```Bash
git stash       # 退避
git stash pop   # 戻す
```
