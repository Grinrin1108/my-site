# 💻 Windowsコマンドプロンプト便利集

Windowsでの作業を効率化するコマンド集です。

## 🌐 ネットワーク・接続確認

### IPアドレスの詳細情報を確認する
* 自分のPCのIPアドレスやMacアドレス、DNSサーバーを一括で確認します。
```
ipconfig /all
```

### 特定のポートを使っている犯人（プロセス）を捜す
* 「ポート8080が既に使用されています」とエラーが出たときに便利です。
```
netstat -ano | findstr :8080
```

### ネットワークの経路を調査する
* 接続先までどこを経由しているか確認します（通信障害の切り分けに）。
```
tracert google.com
```

### ネットワークをリセットする
* ネットが重い時に使うと良い呪文
```
netsh int ip reset
netsh winsock reset
ipconfig/flushdns

```

## 📂 ファイル・ディレクトリ操作

### フォルダ構成を視覚的に表示する
* /f をつけるとファイル名まで、/a をつけると罫線がテキスト表示（コピペ用）になります。
```
tree /f /a
```

### 高機能なファイルコピー（バックアップ用）
* エラーが出ても再試行し、ミラーリングも可能な最強コマンドです。
```
robocopy "C:\Source" "D:\Backup" /MIR /R:3 /W:5
```

### 実行ファイルの場所を特定する
* python や git がどこにインストールされているか調べます。
```
where python
```

## ⚙️ システム管理・診断

### ==danger "管理者権限が必要"== 

* 以下のコマンドは、スタートメニューの「コマンドプロンプト」を右クリックして 「管理者として実行」 を選ぶ必要があります。

### システムファイルの修復
* Windowsの動作が不安定なときに、破損したファイルをスキャンして自動修復します。
```
sfc /scannow
```

### プロセスを強制終了する
* 応答のないアプリを名前やPID（プロセスID）で終了させます。
```
taskkill /F /IM "process_name.exe"
```

## 🛠️ 便利な「ショートカット」

* コマンドこれらを打ち込むだけで、深い階層にある設定画面を一瞬で開けます。

### コントロールパネルを開く 
```
control
```

### システムプロパティ（環境変数など）
```
sysdm.cpl
```

### ネットワーク接続の一覧 
```
ncpa.cpl
```

### プログラムの追加と削除 
```
appwiz.cpl
```

## 📝 記号のクイックリファレンス（リダイレクト）

### > 
* 結果をファイルに書き込む（上書き）
```
ipconfig > ip.txt 
```
### >> 
* 結果をファイルの末尾に追加する 
```
echo 完了 >> log.txt
```

### | 
* 前の結果を次のコマンドに渡す 
```
tasklist | find "chrome"
```

### & 
* 2つのコマンドを連続で実行する 
```
cls & dir
```
